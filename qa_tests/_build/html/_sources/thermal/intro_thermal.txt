Thermal QA Tests and Tutorials
==============================
The set of thermal QA tests exercise PFLOTRAN's TH and GENERAL modes on several 
steady and transient thermal conduction problems. Because these test problems
are meant to test thermal conduction only, the porosity of the porous media has 
been made extremely small (as well as the permeability value) to avoid fluid flow.
